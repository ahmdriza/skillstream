# SkillStream - Online Learning Platform

SkillStream is a modern, feature-rich online learning platform built with Next.js. It offers a premium user experience for discovering, enrolling in, and attending both recorded and live courses.

- **Course Catalog**: Browse a wide range of courses with advanced filtering and search capabilities.
- **Detailed Course Pages**: comprehensive views for Recorded and Live courses, featuring curriculum breakdowns, instructor info, and reviews.
- **User Dashboard**: Personalized dashboard to track progress, view enrolled courses, and manage certifications.
- **Live Learning**: Integrated support for live sessions and scheduling.
- **Authentication**: Secure user authentication (Login, Register, Password Recovery).
- **Responsive Design**: Fully responsive UI ensuring a seamless experience across all devices.
- **Dark Mode**: Built-in dark mode support using Tailwind CSS.

    ```bash

    yarn install
    ```

    Visit [http://localhost:3000](http://localhost:3000) in your browser.
